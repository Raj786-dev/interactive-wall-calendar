# Interactive Wall Calendar

Frontend Engineering Challenge submission for an interactive wall calendar component.

## Features
- Wall calendar inspired UI
- Hero image section
- Month navigation
- Date range selection
- Notes for selected dates/ranges
- Monthly memo section
- Local storage persistence
- Responsive layout
- Dark mode toggle

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Run Locally
```bash
npm install
npm run dev
